TALK DECKは、1台のスマートフォンを回しながら遊ぶ会話カードゲームです。 初対面、友達、親友のセッションモードで、会話の深さや流れが自然に変わります。

無理な行動や危険な行為は避け、周囲に配慮してお楽しみください。
